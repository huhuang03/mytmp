The project template util.

* Install
- clone this project.
- run `npm instal`, `npm build`
- run `npm link`

* Usage
#+BEGIN_SRC 
mytmp -n project_name -t template_name [-o outFolder].
#+END_SRC

** For example
#+BEGIN_SRC 
mytmp -n t_cmake -t cmake
#+END_SRC

will create
#+BEGIN_SRC
- t_cmake
    - build
    - src
        - CMakeList.txt
        - main.cc
#+END_SRC
